# Joguinho basico usando a biblioteca NCURSES e C++

## buid e execução

- Para rodar o jogo já compilado basta usar o comando `./output/main` via terminal.
- Para compilar o projeto basta entrar na pasta do projeto e executar o comando `g++ main.cpp -lncurses -ltinfo -o output/main`.


OBS: O Projeto so roda em sistemas linux e no windows via WSL, porque a biblioteca NCURSES é uma biblioteca para Linux, para rodar no Windows você pode usar a biblioteca **PDCurses**
Para saber mais sobre a biblioteca PDcurses viste o link: https://pdcurses.org/
Tutorial de como instalar a biblioteca PDcurses no Windows: https://terminalroot.com.br/2024/07/como-instalar-a-pdcurses-a-ncurses-no-windows.html
